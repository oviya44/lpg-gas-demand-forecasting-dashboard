# LPG Gas Demand Forecasting Dashboard

## Overview

This project predicts LPG gas demand using historical data and displays insights through an interactive dashboard.

## Features

- LPG demand forecasting
- Data visualization
- Trend analysis
- Interactive dashboard

## Technologies Used

- Python
- Pandas
- Machine Learning
- Dashboard Tools
